#  Portafolio de Análisis de Datos

Bienvenido a mi **Portafolio de Análisis de Datos**.

Este repositorio reúne distintos casos de negocio diseñados para simular tareas reales que realizaría un Analista de Datos.

Cada proyecto sigue un flujo completo de trabajo: consultas SQL, exploración de datos, visualización y recomendaciones para el negocio.

---

##  Estructura del repositorio

Cada carpeta representa un caso de negocio independiente.

```
01-category-revenue-analysis/
...
```

Cada proyecto incluye:

- Escenario de negocio
- Problema a resolver
- Consultas SQL
- Análisis en Excel
- Gráficas
- Resumen ejecutivo
- Recomendaciones
