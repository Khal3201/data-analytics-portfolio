# Data Analytics Portfolio

Welcome to my **Data Analytics Portfolio**.

This repository contains a collection of business case studies designed to simulate real-world tasks performed by a Data Analyst.

Each project follows a complete analytical workflow, including SQL queries, data exploration, visualization, and business recommendations.

---

## Repository Structure

Each folder represents an independent business case.

```
01-category-revenue-analysis/
...
```

Every project includes:

- Business scenario
- Business question
- SQL queries
- Excel analysis
- Charts and visualizations
- Executive summary
- Business recommendations
